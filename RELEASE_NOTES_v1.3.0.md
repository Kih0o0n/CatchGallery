# 캐치갤러리 v1.3.0

여러 화면 크기에서 더 편하게 그리고 감상할 수 있도록 기기 호환성, 입력 안정성과 화면 성능을 다듬었습니다. 새로운 카테고리와 제시어도 대폭 추가했습니다.

## 주요 변경 사항

- 스마트폰·태블릿·PC 화면에 맞춘 전시장, 관리 화면과 그림판 배치 개선
- iPad와 인앱 브라우저에서 발생할 수 있는 그림 입력 끊김·고착 문제 방어
- 점 찍기, 두 손가락 확대·이동, 전체 지우기 확인과 되돌리기 개선
- 이미지 지연 로딩, 제한형 캐시와 안전한 화면 전환으로 성능 개선
- 로그인·회원가입·자동 로그인 프로필 준비 흐름 안정화
- 데이터 표시 보안과 이미지 저장·수정 구조 강화
- 제시어를 999개로 확장하고 6개 카테고리 추가

> 실제 기기와 브라우저 환경에 따라 동작 차이가 있을 수 있습니다.
>
> 문제가 발생하면 앱의 ‘의견 보내기’를 통해 알려주세요.
